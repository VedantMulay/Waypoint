# Waypoint
Waypoint is a Spigot/Paper plugin which helps players navigate from your server.
Players can create and save their waypoints as well as access global waypoints set by the server admins.
A highly useful plugin for survival, lifesteal, RPG and vanilla servers.

## Features
- Custom personal waypoints
- Global server waypoints
- GUI for accessing waypoints
- Highly interactive

## Planned Features
- Multi-Server waypoints and tasks

## Requirements:
- Java 17+
- Spigot/Paper 1.19.4+
- Protocol Lib v5.4.0+
